# 의어 v0.1.1 (eui lang)

오직 '의'로 이루어진 인터프리터

## Syntax

### 1. var/const

```
의! a 10
의!! b 10
```
의! = var 선언

의!! = const 선언

### 2. Operator
#### 2.1. Numeric Operation
```
의! a 20 덧의 10
의! b 20 차의 10
의! c 20 곱의 10
의! d 20 분의 10
의! e 20 남의 10
의! f 반의 10
```
+, -, *, /, % = 덧의, 차의, 곱의, 분의, 남의

-a = 반의 a

#### 2.2 Comparison Operation
```
의! g 20 의 10
의! h 20 늬 10
의! i 20 대의 10
의! j 20 소의 10
의! k 20 대의의 10
의! l 20 소의의 10
```
==, != = 의, 늬

<, <=, >, >= = 대의, 대의의, 소의, 소의의

#### 2.3 Boolean Operation
```
의! m 읭 합의 닁
의! n 읭 교의 닁
의! o 읭 늬의 닁
의! p 부의 읭
```
|, &, ^ = 합의, 교의, 늬의

!a = 부의 a

#### 2.4 Bitwise Operation
```
의! q 20 합의 10
의! r 20 교의 10
의! s 20 늬의 10
의! t 부의 10
```

~a = 부의 a

### 3. if/then/else
```
의? b
의: 의! a a 덧의 10
늬: 의! a a 차의 10
의: 의! a a 곱의 10
```
if, then, else = 의?, 의:, 늬:
if엔 항상 Boolean이 들어감
의:와 늬:는 여러번 사용 가능(다음 if가 올 때 까지)

### 4. while/continue/break
```
의! a 0
의! b 10

의의? a 늬 b
의: 의! a a 덧의 1
의: 계의
```
while, continue, break = 의의?, 계의, 종의(예시엔 나오지 않음)

의의?는 사실상 계의를 통해 돌아갈 수 있는 의?와 다를 바 없음

일반적인 while처럼 쓰고 싶다면 의: 계의 명시적(code block 존재하지 않음)

늬: 계의 처럼 응용 할 수도 있다 (물론 일반적인 while의 개념에서 벗어나게 됨)

종의는 의:를 비활성화, 늬:를 활성화, 계의 비활성화

### 5. Value
```
의! a 10
의! b 10.5
의! c "string"
의! d 읭
의! e 닁
```
(현재 string 타입은 사용 불가)

true, false = 읭, 닁
